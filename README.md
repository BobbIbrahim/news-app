# News List App (Android Kotlin Lab)

A simple Android app built with **Kotlin** that displays a list of news items using **RecyclerView**.  
Each item shows an **icon**, **title**, and **description**, with a clean **Card UI** and a **divider** between items.

---

## Features

- Single Activity app 
- News feed using **RecyclerView**
- Custom row layout (`item_news.xml`)
- **CardView** UI for a modern look
- Divider line between items
- Different title/description + different icon per item (bonus)
- Icons tinted with realistic colors (category-based) 
- Press feedback (ripple effect)
